Description

NoelPrint est une application Windows développée en C# permettant de prendre des screenshots et de les uploader automatiquement sur NoelShack, en copiant à la volée le lien généré dans le presse papier.

Il est possible de lancer automatiquement NoelPrint au démarrage de Windows, et une option additionnelle permet d'exporter l'historique des screenshots au format HTML.

Bugs connus

L'exécutable est compilé avec Microsoft .NET Framework 4.5, si vous avez une version antérieure NoelPrint risque de se lancer et de crasher avec le message d'erreur suivant : 

System.TypeLoadException: Impossible de charger le type 'System.Runtime.CompilerServices.ExtensionAttribute' à partir de l'assembly 'mscorlib'

Veillez à installer Microsoft .NET Framework 4.5 pour que NoelPrint fonctionne correctement.

Utilisation

L'utilisation de NoelPrint se résume à trois raccourcis permettant respectivement de prendre une capture globale, une capture de la fenêtre active, ou de capturer une zone personnalisée sur l'écran.

Ctrl + 2 (é)
Capture globale
Ctrl + 3 (")
Capture de la fenêtre active
Ctrl + 4 (')
Capture d'une zone personnalisée

Lors de l'activation d'une de ces fonctionnalitées, une bulle de notification sera affichée en bas à droite indiquant le succès de l'upload. Le lien de l'image uploadée sera alors automatiquement copié dans le presse papier, et un clic sur la bulle de notification l'ouvrira dans votre navigateur préféré !

NoelPrint ajoute également un icône dans la zone de notifications, qui propose les fonctionnalités précédentes lors d'un clic droit.

Le menu contextuel de l'icône NoelPrint permet également d'autres fonctionnalités.

Sauvegarder l'historique
Sauvegarde l'historique des 32 derniers screenshots dans un fichier HTML
Lancer au démarrage
Permet de lancer ou non NoelPrint au démarrage de Windows via une case à cocher dans le menu contextuel
Quitter
Exécute les tâches définies par l'application associées à la libération ou à la redéfinition des ressources non managées
